# Advanced SQL Portfolio

Advanced SQL portfolio featuring CTEs, Window Functions, analytical rankings, segmentation, and football analytics projects.

## About

This repository contains a collection of advanced SQL exercises completed using a football league database.

The project focuses on:

- Common Table Expressions (CTEs)
- Window Functions
- RANK(), DENSE_RANK(), and ROW_NUMBER()
- PARTITION BY
- CASE expressions
- Analytical ranking systems
- Football performance analysis
- Moneyball-style metrics

## Dataset

The exercises were developed using a football league database containing:

- Teams
- Players
- Goals
- Assists
- Clean Sheets
- Market Values
- Team Budgets
- League Standings

## SQL Topics Covered

### Window Functions
- Global rankings

- League averages

## Final Project

The repository includes a final analytical model combining:

- CTEs
- DENSE_RANK()
- RANK()
- PARTITION BY
- CASE expressions
- Custom performance metrics

to evaluate player performance, market efficiency, and team hierarchy.
